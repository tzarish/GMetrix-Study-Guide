JavaScript Events

Event Objects

The JavaScript Event object is an object automatically generated by the browser to store information about an event that has occurred on a web page, such as a user click or key press. This object is passed as an argument to event handler functions and provides properties to control the event's behavior and access details about the user's action. The event object is often written as "e", "event", or "element".

Event Propagation

Bubbling vs. Cascading

In JavaScript, event bubbling and capturing are the two main phases of event propagation in the DOM (Document Object Model), describing the direction in which an event travels through nested elements.

Explanation:

Event bubbling occurs when event handlers are invoked when one element is nested to another element that is part of the same event.

Event cascading occurs when an element is accessed inside of another element through event handlers.

An event capture occurs when an event moves from the outermost element of a group of elements to a target element.
